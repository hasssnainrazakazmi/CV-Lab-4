CV-Lab-4: Computer Vision Implementations
Project Overview
This repository contains a collection of Python-based image processing and computer vision implementations using OpenCV, developed across several laboratory sessions. The projects range from fundamental image manipulation to advanced applications like privacy masking and disease detection in agriculture.
